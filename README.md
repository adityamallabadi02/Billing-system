# Billing-system
This project is a console-based Billing System developed using the C programming language. It is designed to manage customer billing records efficiently using file handling techniques.
The system allows users to add new bills, store customer purchase details, and display all stored billing records. All data is saved permanently in a file, making it a simple example of a real-world data management application.

🚀 Features

✅ Add new customer bill details
✅ Store billing data in a file
✅ Display all saved bills
✅ Automatic total bill calculation
✅ Menu-driven user interface
✅ Persistent storage using files

🛠️ Concepts Used

This project demonstrates important C programming concepts such as:
Structures (struct)
Functions
File handling (fopen, fprintf, fscanf, fread, fwrite)
Menu-driven programming
String input handling
Loops and conditional statements
